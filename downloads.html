Versioning System (VertigoMaj#Min)

Vertigo releases follow a VertigoMaj#Min numbering system, where:

  • Maj indicates a major update with significant new features or breaking changes.
  • Min indicates a minor update or hotfix, containing bug fixes or small improvements.

For example, Vertigo2#2 would signify the second major version with its second minor update. The download links above always point to the latest release on GitHub, so you'll always get the most current Maj#Min available.

Initial Setup Instructions

After downloading your chosen `.zip` file, follow these steps to get started:

  • Extract the contents of the `.zip` file to a dedicated folder for Vertigo (e.g., C:\Vertigo on Windows, or ~/vertigo on Linux).
  • This folder will contain vertigo.exe (or vertigo) and dizzypack.exe (or dizzypack), along with the libs/ directory.
  • Add the Vertigo folder to your System PATH: This crucial step allows you to run vertigo and dizzypack commands from any directory in your terminal or command prompt. (Search online for "how to add to PATH on [your operating system]" for detailed instructions.)
  • Restart your terminal/command prompt after modifying the PATH to ensure changes take effect.

Once set up, you can start writing your .vtgo programs and managing libraries. Refer to the Get Started page for your first program!
