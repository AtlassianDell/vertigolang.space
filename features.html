Core Capabilities

Vertigo is built with a powerful and explicit feature set, giving developers unparalleled control and clarity.

  • Dynamic Stacks

    Create and manage multiple, independent data stacks, offering unparalleled flexibility in how your program handles information. This allows for complex data structures and efficient memory use.

  • Dedicated Registers

    Leverage explicit registers like ODA (Output Data Accumulator), IDA (Input Data Accumulator), CLI (Current Loop Iteration), and LTM (Loop Termination Max) for direct data access and fine-grained control over program state.

  • Precise Flow Control

    Navigate your code with ultimate precision using named labels and a robust set of conditional jump instructions. This includes JUMPEQ (Jump if Equal), JUMPGT (Jump if Greater Than), JUMPLT (Jump if Less Than), and JUMPNEQ (Jump if Not Equal), allowing for complex branching logic.

  • Comprehensive Data Operations

    Seamlessly perform a wide range of operations on both numerical and string data. This includes arithmetic functions (MATH), string concatenation (CONCAT), calculating string lengths (STRLEN), and robust string comparisons (STRCMP).

  • Modular Subroutines

    Enhance code reusability and maintainability by organizing your logic into easily callable subroutines. This promotes clean, structured programming practices.

  • External Integration

    Expand your projects by importing external Vertigo libraries using the BRING instruction, or integrate directly with existing Python functionalities using the IMPORT instruction for broader capabilities. These libraries are conveniently managed via dizzypack.

  • Immutable Constants

    Define values using the IM instruction that are guaranteed to remain constant throughout your program's execution, ensuring data integrity and predictable behavior.

  • System Interactions

    Interface directly with the operating environment and hardware through predefined system interrupts (INT). This provides low-level access for specialized tasks and system control.

  • Stack Manipulation

    Advanced stack manipulation instructions like SWAP (swap top two items), PICK (duplicate an item from deeper in the stack), PPICK (duplicate and remove original), CLEAR (empty stack), ROT (rotate top three items), and RROT (reverse rotate) provide powerful data rearrangement capabilities.

  • Logical Operations (OPS)

    Perform boolean logic with the OPS instruction, supporting operations like AND, OR, NOT, EQUAL, and NEQUAL for complex conditional evaluations.
